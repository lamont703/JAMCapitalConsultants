JAM Credit Assessment Module

This module provides a complete credit assessment quiz that can be embedded anywhere on your website with just a few lines of code.

1. Basic Usage

Include the script and create a button:

<script src="CreditAssessmentModule.js"></script> <script> // The module auto-initializes, just create a button document.body.innerHTML += JAMCreditAssessment.createButton('Start Your Credit Assessment'); </script>

Or use a simple button with onclick:

<button onclick="JAMCreditAssessment.open()">Take Credit Quiz</button>

2. Custom Configuration

Configure the module with custom completion and start handlers:

<script> JAMCreditAssessment.init({ onStart: function() { console.log('Quiz started!'); // Track analytics, etc. }, onComplete: function(data) { console.log('Quiz completed:', data); // Send to your backend, show success message, etc. alert('Thanks! We will send your personalized credit guide to ' + data.email); } }); </script>

3. Multiple Buttons

You can have multiple buttons on the same page:

<button onclick="JAMCreditAssessment.open()">Header Button</button> <button onclick="JAMCreditAssessment.open()">Footer Button</button> <button onclick="JAMCreditAssessment.open()">Sidebar Button</button>

4. Programmatic Control

You can also control the quiz programmatically:

// Open the quiz JAMCreditAssessment.open(); // Close the quiz JAMCreditAssessment.close(); // Check if initialized if (JAMCreditAssessment.initialized) { // Module is ready }

5. Integration Examples

Easy integration with popular frameworks:

WordPress:

// Add to your theme's functions.php function add_jam_credit_assessment() { wp_enqueue_script('jam-credit-assessment', get_template_directory_uri() . '/js/CreditAssessmentModule.js', array(), '1.0.0', true); } add_action('wp_enqueue_scripts', 'add_jam_credit_assessment'); // In your template: <button onclick="JAMCreditAssessment.open()">Start Credit Assessment</button>

React:

useEffect(() => { // Load the script const script = document.createElement('script'); script.src = '/js/CreditAssessmentModule.js'; document.body.appendChild(script); }, []); // In your component: <button onClick={() => window.JAMCreditAssessment.open()}> Start Credit Assessment </button>

GoHighLevel:

<!-- Add to your funnel page custom code section --> <script src="https://yoursite.com/js/CreditAssessmentModule.js"></script> <button onclick="JAMCreditAssessment.open()" class="btn btn-primary"> Start Credit Assessment </button>
